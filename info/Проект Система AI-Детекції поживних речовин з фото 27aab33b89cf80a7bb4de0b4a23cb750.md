# Опис системи AI-Brocalorie

## Що це?

**AI-Brocalorie** — це AI-система для аналізу фото їжі, яка автоматично визначає поживну цінність страв, включаючи калорії, макронутрієнти (білки, жири, вуглеводи), мікронутрієнти (вітаміни, мінерали) та потенційні алергени. Сервіс працює через API, дозволяючи користувачам отримувати детальну інформацію про їжу, просто завантаживши фото.

---

## Основний функціонал

- **Аналіз фото їжі**: Завантажуйте зображення страви, і система розпізнає інгредієнти та їх поживну цінність.
- **Детальний розклад поживних речовин**: Отримуйте дані про калорії, білки, жири (включаючи насичені, трансжири тощо), вуглеводи (клітковина, цукри, крохмаль), воду, холестерин, а також вітаміни та мінерали.
- **Виявлення алергенів**: Система ідентифікує алергени (наприклад, глютен, молочні продукти), допомагаючи користувачам уникати небезпечних продуктів.
- **API-доступ**: Інтеграція через API (наприклад, `curl -X POST -F image=@apple.jpg http://nutriscan.radarme.com.ua/analyze/`) для легкого використання в додатках чи сервісах.
- **JSON-вивід**: Результати повертаються у структурованому форматі JSON із детальними даними про страву та її складові.

---

## Для чого?

- **Фітнес та здорове харчування**: Допомагає користувачам відстежувати калорії та макронутрієнти для досягнення фітнес-цілей (схуднення, набір м’язів, підтримка форми).
- **Дієтичні потреби**: Підходить для людей із харчовими обмеженнями (алергії, дієти без глютену чи лактози).
- **Зручність**: Спрощує аналіз харчування, усуваючи необхідність вручну вводити дані про їжу.
- **Інтеграція**: API дозволяє інтегрувати сервіс у фітнес-додатки, трекери харчування чи ресторанні системи.

---

## Цільова аудиторія

- Люди, які стежать за харчуванням (фітнес-ентузіасти, спортсмени).
- Особи з дієтичними обмеженнями чи алергіями.
- Розробники, які хочуть додати аналіз харчування до своїх додатків.
- Ресторани чи сервіси доставки їжі для автоматичного надання клієнтам інформації про поживну цінність.
